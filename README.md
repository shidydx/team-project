## Summary
The domain of our project is news analysis and media literacy. Our software allows
users to input a news topic and view the output concise summary of both left and
right leaning perspectives on the given topic. The purpose of this application is
to help users understand and recognize political bias, compare the difference in
viewpoints on the same issue, and make a balanced judgement for themselves on
current events.

## APIs
- API Name: https://newsapi.org/
    - Main service provided:
        - Fetches recent and relevant news articles from online sources and blogs worldwide
        - Supports searching by keywords, domain, language and date range
    - **Status**: Tested with OkHttp in Java.
- API Name: https://platform.openai.com/docs/models/gpt-5-mini
    - Main service provided:
        - Generate summaries and analyses in natural language.
    - Status: Tested with OkHttp in Java.
